Ukigumo

Life with continious testing

Ukigumo is yet another Continuous Testing toolchain.

Ukigumo tool kit composed by following stuff.

Ukigumo Server →

Storage server of testing result. Server provides full of RESTful APIs.

Ukigumo Agent →

Ukigumo Agent is test runner server. You can enqueue the test request by git post-receive hook. Ukigumo Agent server provides RESTful APIs.

Ukigumo Client →

CLI client for ukigumo-server.
